# 134. Gas Station

## 문제 분석

- gas station을 시계방향으로 순회할 수 있는 시작 위치 구하기, 없으면 -1 리턴
- 정답이 하나인 것 보장
- n == gas.length == cost.length
- 1 <= n <= 10^5
- 0 <= gas[i], cost[i] <= 10^4

## 풀이

- n이 9999이고 gas와 cost의 값이 마지막을 제외하고 모두 0이나 1로 채워져 있는 경우가 있어 Time Exceeded 에러가 났음
  - `if(gas[i]<=cost[i]) continue;` 추가
- gas station이 한 군데인데 gas와 cost의 값이 같은 경우
  - `if(gas.length==1 && gas[0]==cost[0]) return 0;` 추가

``` java

class Solution {
    public int canCompleteCircuit(int[] gas, int[] cost) {
         if(gas.length==1 && gas[0]==cost[0]) return 0; 
        
         for(int i=0;i<gas.length;i++){

            if(gas[i]<=cost[i]) continue;

            if(travel(i, gas, cost)){
                return i;
            }
         }

         return -1;

    }

    private static boolean travel(int start, int[] gas, int[] cost){
        
        int tank=gas[start];
        int now=start;
        int next=(start+1)%gas.length;

        do {
            tank-=cost[now];
            if(tank<0) return false;
            tank+=gas[next];
            now=next;
            next=(next+1)%gas.length;
        } while(now!=start);

        return true;
    }
}

```
