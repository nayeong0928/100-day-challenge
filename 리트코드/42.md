# Trapping Rain Water

## 문제

- 비가 온 후 틈새 사이에 갇힌 물의 양을 구하기
- n == height.length
- 1 <= n <= 2 * 10^4
- 0 <= height[i] <= 10^5

## 풀이

- max: 가장 높은 높이를 가진 인덱스
- h: 현재 체크한 높이 중에서 가장 높은 높이
- 0 ~ max, 배열 마지막 값 ~ max 까지 순회하면서 물의 양을 구한다
  - height[i] > h 인 경우: h 갱신
  - height[i] < h 인 경우: 물의 양에 (h-height[i]) 더해주기
- 배열의 길이가 20000 이하이기 때문에 max를 구할 때 한 번, 0 ~ max, height.length-1 ~ max 까지 세 번 순회해도 성능에 문제 없음

``` java

class Solution {
    public int trap(int[] height) {
        int ans=0;
        int max=0;

        for(int i=0;i<height.length;i++){
            if(height[max]<height[i]){
                max=i;
            }
        }

        int h=height[0];

        for(int i=0;i<max;i++){
            if(height[i]>h){
                h=height[i];
            }
            else if(height[i]<h){
                ans+=(h-height[i]);
            }
        }

        h=height[height.length-1];
        
        for(int i=height.length-1;i>max;i--){
            if(height[i]>h){
                h=height[i];
            }
            else if(height[i]<h){
                ans+=(h-height[i]);
            }
        }

        return ans;
    }
}

```
