# 380. Insert Delete GetRandom O(1)

## 문제 분석

- insert(val) : 값이 있으면 false 리턴, 값이 없으면 val을 삽입하고 true 리턴
- remove(val) : 값이 없으면 false 리턴, 값이 있으면 val을 삭제하고 true 리턴
- getRandom() : 원소들을 같은 확률로 랜덤하게 리턴함, 이 메소드가 호출될 때에는 최소한 1개의 원소가 있음
- -2^31 <= val <= 2^31 - 1

## 풀이 (Time Limit Exceeded)

- 해시셋을 고려했었지만 해시셋은 순서가 없기 때문에 getRandom() 에서 같은 확률로 리턴된다는 점을 보장할 수 없음
- 삽입/삭제가 자주 일어난다는 점을 고려하여 LinkedList 선택
- idx: 랜덤값을 어디까지 출력했는지 저장. 삭제/삽입이 일어나는 경우 갱신해준다.

``` java

class RandomizedSet {

    static LinkedList<Integer> list;
    static int idx;

    public RandomizedSet() {
        list=new LinkedList<>();
        idx=0;
    }
    
    public boolean insert(int val) {
        if(search(val)){
            return false;
        }

        list.add(val);
        idx=(idx+1)%list.size();
        return true;
    }

    private boolean search(int val){
        for(int i=0;i<list.size();i++){
            if(list.get(i)==val){
                return true;
            }
        }

        return false;
    }
    
    public boolean remove(int val) {
        if(!search(val)){
            return false;
        }

        list.remove(Integer.valueOf(val));
        
        if(list.size()==0){
            idx=-1;
        }
        else{
            idx=(idx+1)%list.size();
        }
        return true;
    }
    
    public int getRandom() {
        
        int val=list.get(idx);
        idx=(idx+1)%list.size();
        return val;
    }
}

```
