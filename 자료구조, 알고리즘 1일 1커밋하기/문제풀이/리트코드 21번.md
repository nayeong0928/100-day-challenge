# Merge Two Sorted List


## 문제 정의

- list1, list2
  - 모두 정렬되어 있음
  - 리스트 길이는 0~50 => null도 가능
  - 리스트 값 범위: -100 ~ 100
  - 둘 다 오름차순으로 정렬됨
- 정렬된 리스트 합치기
- 합친 linked list의 head 리턴


## 구현

- dummyHead: 결과 리스트의 head 저장을 위한 더미 헤드
- curr: 리스트 추가를 위한 노드
- 다른 리스트가 null이 되어 while문을 빠져나왔지만 아직 방문하지 않은 노드가 있을 수 있으므로 while문 이후에 아직 null이 아닌 리스트 노드가 있으면 추가해준다.

```java

public ListNode mergeTwoLists(ListNode list1, ListNode list2) {
        
        if(list1==null && list2==null) return null;
        
        ListNode dummyHead=new ListNode(0, null);
        ListNode curr=dummyHead;

        while(list1!=null && list2!=null){
            if(list1.val<=list2.val){
                curr.next=list1;
                list1=list1.next;
            }else {
                curr.next=list2;
                list2=list2.next;
            }
            curr=curr.next;
        }

        if(list1!=null){
            curr.next=list1;
        }

        if(list2!=null){
            curr.next=list2;
        }
        
        return dummyHead.next;
    }
```
