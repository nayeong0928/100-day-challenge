# 재배열 문제

정수 배열이 주어졌을 때 짝수가 먼저 나오도록 재배열해 보라.  

## 문제 분석

- 짝수가 먼저 나오기만 하면 되고 짝수/홀수 내에서 오름차순이나 내림차순을 할 필요는 없음
- 인덱스 0부터 시작해서 배열을 순회하면서 홀수인 경우 -> 배열의 맨 뒤로 보내줌
- 시간복잡도: O(N)
- 추가 공간복잡도: O(1)


```java

    public static void evenOdd(int[] arr) {
        int i = 0;
        int j = arr.length - 1;

        while (i < j) {
            if (arr[i] % 2 == 0) {
                i++;
            } else {
                swap(arr, i, j);
                j--;
            }
        }
    }

    public static void swap(int[] arr, int i, int j) {
        int tmp = arr[i];
        arr[i] = arr[j];
        arr[j] = tmp;
    }

```
