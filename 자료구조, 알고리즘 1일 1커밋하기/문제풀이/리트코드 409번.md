# Longest Palindrome

## 문제 분석

- 입력된 문자열에서 가장 긴 palindrome의 길이 구하기
- 부분 palindrome은 연속된 문자열이 아니어도 됨
- 대소문자 구분
- 문자열의 길이는 1 ~ 2000

## 구현

- 원본 문자열에서 연속된 값이 아니어도 됨 -> 글자들의 숫자만 체크
- 문자열의 길이가 짝수 길이인 경우: palindrome의 각 글자가 짝수개 있어야 한다
- 문자열의 길이가 홀수 길이인 경우: palindrome의 각 글자가 짝수개 있어야 하는 건 마찬가지. 그렇지만 대칭의 중심 위치에 글자 한 개가 더 올 수 있다
- key만 저장되면 되기 때문에 HashSet 사용

### 로직
- 해시 결과 없는 경우 : 해시 추가
- 해시 결과 있는 경우 : 삭제, result+=2
- 문자열 끝까지 탐색 후 해시에 남는 값 있다면 -> result++

``` java

    public int longestPalindrome(String s) {
        Set<Character> hash=new HashSet<>();
        int result=0;
        
        for(int i=0; i<s.length(); i++){
            char c=s.charAt(i);
            if(hash.contains(c)){
                hash.remove(c);
                result+=2;
            }
            else {
                hash.add(c);
            }
        }

        if(hash.size()>0) {
            result++;
        }

        return result;
    }
```
