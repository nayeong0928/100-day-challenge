# Jump Game

## 문제 분석
- 0번 인덱스에 위치
- nums[i]=i번째에서 점프 가능한 최대수 
- 맨 마지막에 도달 가능하면 true, 아니면 false
- nums의 길이 1 ~ 10000
- nums[i] 0 ~ 100000

      
## 구현

-    visit 배열
-    nums[i]에서 방문 가능한 곳들 표시
-    visit이 true인 곳에서 방문 가능한 곳 표시
-    맨 마지막 인덱스까지 체크
  


```java


    public boolean canJump(int[] nums) {
        
        if(nums.length==1) return true;

        boolean[] visit=new boolean[nums.length];
        int jump=0;
        visit[0]=true;

        for(int i=0;i<visit.length-1; i++){
            
            if(visit[i]) {
                int possible=nums[i]+i;
                
                while(jump<=possible && jump<nums.length){
                    visit[jump++]=true;
                }
            }
        }

        return visit[visit.length-1];
    }

```
