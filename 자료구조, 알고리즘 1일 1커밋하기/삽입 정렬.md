# 삽입 정렬

정렬된 배열의 크기를 1부터 n까지 하나씩 늘려가는 정렬 방법. (n=배열의 길이)  
정렬된 배열의 크기를 늘릴 때 새로운 배열의 자리를 찾아주면서 정렬한다.

- 최악 시간복잡도: O(n^2)
- 평균 시간복잡도: O(n^2)
- 최선 시간복잡도: O(n)
  
## 구현

- i: 정렬된 배열의 길이
- j: 새로운 요소의 자리를 찾아주는 역할

예를 들어 arr={1,2,4,5,3} 에서 i=4일 때,  
j=3 -> {1,2,4,3,5}  
j=2 -> {1,2,3,4,5}  
j=1 -> arr[1]인 2는 arr[i]인 3보다 작기 때문에 while문이 멈추면서 배열이 정렬된다.

```java

    private static void insertionSort(int[] arr) {

        for (int i = 1; i < arr.length; i++) {
            int item = arr[i];
            int j = i - 1;

            while (j >= 0 && arr[j] > item) {
                int tmp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = tmp;
                j--;
            }
        }
    }

```
