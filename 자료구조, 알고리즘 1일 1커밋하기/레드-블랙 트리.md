# 레드-블랙 트리

- 레드/블랙 두 색을 사용해서 균형을 맞추는 이진 검색 트리

### 특성
- 루트는 블랙이다
- 모든 리프 노드는 블랙이다
- 루트로부터 임의의 리프 노드에 이르는 경로에 레드 2개가 연속으로 올 수 없다
- 루트에서 임의의 리프 노드에 이르는 경로에서 만나는 블랙의 수는 모두 같다

### 수선
1. 부모 노드가 블랙인 경우: 레드인 상태로 삽입
2. 부모 노드가 레드인 경우: 오른자식일 경우 좌회전, 왼자식일 경우 우회전 후 부모와 색을 바꿔줌

### 성능
- 검색: O(log n)
- 삽입: O(log n) -> 최악의 경우 불균형이 루트 노드까지 올라가더라도 O(log n)을 여러번 반복하기 때문
- 삭제: O(log n)
