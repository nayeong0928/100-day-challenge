# 물통

## 문제 분석

- A, B, C는 1 ~ 200 범위
- 시작: A, B는 비어있고 C는 물이 다 차 있음
- A가 비어 있을 때 C에 담겨 있을 수 있는 물의 양 모두 구하기
- 한 물통이 비거나, 다른 물통이 다 찰 때까지 물을 부을 수 있음 (다른 경우는 불가능)

## 풀이 (오답)

- from -> to 로 물을 붓는 경우를 dfs로 계산
- from -> to 로 두번 물을 부을 수도 있음

```java

import java.util.HashSet;
import java.util.Scanner;

public class P2251 {

    static final int A = 0;
    static final int B = 1;
    static final int C = 2;
    static HashSet<Integer> hash;

    public static void main(String[] args) {
        hash = new HashSet<>();
        int[] arr = new int[3];
        int[] cap = new int[3];

        Scanner sc = new Scanner(System.in);
        cap[A] = sc.nextInt();
        cap[B] = sc.nextInt();
        cap[C] = sc.nextInt();

        // 시작상태: A,B 비어있고 C는 차있음
        arr[C] = cap[C];
        dfs(C, A, arr, cap);
        dfs(C, B, arr, cap);

        for (int i = 1; i <= 200; i++) {
            if (hash.contains(i)) {
                System.out.print(i + " ");
            }
        }

    }

    private static void dfs(int from, int to, int[] arr, int[] cap) {

        // from -> to 물붓기
        int size = arr[from] + arr[to] - cap[to];
        if (arr[from] < size) {
            size = arr[from];
        }
        arr[from] -= size;
        arr[to] += size;

        if (arr[A] == 0) {
            hash.add(arr[C]);
            return;
        }

        for (int f = A; f <= C; f++) {
            for (int t = A; t <= C; t++) {
                if (f == t) {
                    continue;
                }
                
                if (cap[t] > arr[t]) {
                    dfs(f, t, arr, cap);
                }
            }
        }
    }
}


```
