# 특정 거리의 도시 찾기

## 문제 분석

- 단방향, 도시의 개수가 도로의 개수보다 훨씬 많음 (2 ≤ N ≤ 300,000, 1 ≤ M ≤ 1,000,000)
- 리스트 배열 형태로 도시에서 방문 가능한 도시 리스트 저장
- 제일 처음 방문할 도시가 정해져 있기 때문에 큐에 집어넣고 시작

## 풀이

1. X를 queue에 넣음
2. 연결 리스트에 있는 도시 방문
3. visited 이면 방문 안 함
4. visited가 false인 경우 방문, 도시 dist 정보 갱신
5. dist 체크해서 K 값을 가진 도시 출력, 없으면 -1 출력

### 코드

```java

import java.util.ArrayList;
import java.util.LinkedList;
import java.util.Queue;
import java.util.Scanner;

public class P18352 {

    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);
        int N = sc.nextInt();
        int M = sc.nextInt();
        int K = sc.nextInt();
        int X = sc.nextInt();
        ArrayList<Integer>[] map = new ArrayList[N + 1];

        for (int i = 1; i <= N; i++) {
            map[i] = new ArrayList<Integer>();
        }

        // 경로 체크
        for (int i = 0; i < M; i++) {
            int A = sc.nextInt();
            int B = sc.nextInt();
            map[A].add(B);
        }

        String res = BFS(map, K, X);

        if (res == "") {
            System.out.println("-1");
        } else {
            System.out.println(res);
        }
    }

    private static String BFS(ArrayList<Integer>[] map, int K, int X) {

        Queue<Integer> queue = new LinkedList<>();
        int[] dist = new int[map.length];
        boolean[] visited = new boolean[map.length];

        queue.add(X);
        visited[X] = true;

        while (!queue.isEmpty()) {
            int q = queue.poll();

            for (int m : map[q]) {
                if (!visited[m]) {
                    queue.add(m);
                    dist[m] = dist[q] + 1;
                    visited[m] = true;
                }
            }

        }

        StringBuilder sb = new StringBuilder();

        for (int i = 1; i < dist.length; i++) {
            if (dist[i] == K) {
                sb.append(i + "\n");
            }
        }

        return sb.toString();
    }
}


```
