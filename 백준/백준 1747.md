# 소수 & 펠린드롬

## 문제 분석

- 펠린드롬수: 어떤 수와 그 수의 숫자 순서를 뒤집은 수가 일치하는 수
- N보다 크거나 같고, 소수이면서 팰린드롬인 수 중에서, 가장 작은 수를 구하기
- N의 범위는 1 ≤ N ≤ 1,000,000

## 풀이

- prime(): 소수 체크하는 함수
- pelindrome(): 펠린드롬수 체크하는 함수
- 펠린드롬수는 한자리수 (2,3,5 등)도 포함한다 소수 & 펠린드롬

```java

import java.util.Scanner;

public class sol1747 {

    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);
        int N = sc.nextInt();
        int ans = pelindrome(N, prime());
        System.out.println(ans);
    }

    private static boolean[] prime() {

        int N = 10000000;
        boolean[] arr = new boolean[N + 1];

        arr[0] = true;
        arr[1] = true;

        for (int i = 2; i < arr.length; i++) {
            for (int j = 2; j < arr.length; j++) {
                int n = i * j;
                if (n > N)
                    break;
                arr[n] = true;
            }
        }

        return arr;
    }

    private static int pelindrome(int N, boolean[] arr) {

        for (int i = N; i < arr.length; i++) {
            if (check(i) && !arr[i]) {
                return i;
            }
        }

        return -1;
    }

    private static boolean check(int num) {

        String str = String.valueOf(num);
        int start = 0;
        int end = str.length() - 1;

        while (start < end) {
            if (str.charAt(start) != str.charAt(end))
                return false;
            start++;
            end--;
        }

        return true;
    }

}
```
