# 줄 세우기

## 문제 해석

- N명의 학생들을 키 순서대로 줄 세우기
- M: 키를 비교한 횟수
- A가 B의 앞에 선다
- 답이 여러개인 경우 아무거나 출력 가능
- 1 ≤ N ≤ 32,000
- 1 ≤ M ≤ 100,000

## 풀이

- dist[i]: i번 학생의 앞에 있어야 하는 학생의 수
- dist[i]=0 인 경우 queue에 넣는다
- queue: 줄세우기를 마친 학생
- queue에서 꺼낸 다음, 그 다음에 올 수 있는 학생들의 dist값을 1씩 빼준다

```java

import java.util.ArrayList;
import java.util.LinkedList;
import java.util.Queue;
import java.util.Scanner;

public class P2252 {

    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);
        int N = sc.nextInt();
        int M = sc.nextInt();

        ArrayList<Integer>[] arr = new ArrayList[N + 1];
        int[] dist = new int[N + 1];
        Queue<Integer> queue = new LinkedList<>();
        StringBuilder sb = new StringBuilder();

        for (int i = 1; i < arr.length; i++) {
            arr[i] = new ArrayList<>();
        }

        for (int i = 0; i < M; i++) {
            int A = sc.nextInt();
            int B = sc.nextInt();

            arr[A].add(B);
            dist[B]++;
        }

        for (int i = 1; i < dist.length; i++) {
            if (dist[i] == 0) {
                queue.add(i);
            }
        }

        while (!queue.isEmpty()) {
            int p = queue.poll();
            sb.append(p + " ");

            for (int a : arr[p]) {
                dist[a]--;

                if (dist[a] == 0) {
                    queue.add(a);
                }
            }
        }

        System.out.println(sb.toString());

    }

}

```
