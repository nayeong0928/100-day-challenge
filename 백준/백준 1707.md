# 이분 그래프

## 문제 분석

- 이분 그래프: 그래프의 정점의 집합을 둘로 분할, 집합에 속한 정점끼리는 서로 인접하지 않도록 분할할 수 있는 그래프 
- K: 테스트 케이스의 수 (2 ≤ K ≤ 5)
- V: 정점 (1 ≤ V ≤ 20,000)
- E: 간선 (1 ≤ E ≤ 200,000)

## 풀이 (오답)

- 그래프의 사이클이 존재하면 이분 그래프일 수 없음
- 사이클이 있는지 없는지 DFS로 찾도록 함 -> 한 정점에서 방문할 수 있는 모든 정점들을 방문번호(start)로 grp에 저장
- 모든 정점들이 연결되어 있으면 방문번호가 모두 똑같을 것이기 때문에 NO를 출력하도록 했다
- 반례: 전체 노드가 2개인 상태에서, 둘이 서로 연결되어 있는 경우

```java

import java.util.ArrayList;
import java.util.Scanner;

public class P1707 {

    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);
        int K = sc.nextInt();

        for (int i = 0; i < K; i++) {
            int V = sc.nextInt();
            int E = sc.nextInt();

            ArrayList<Integer> arr[] = new ArrayList[V + 1];
            int[] grp = new int[V + 1];

            for (int j = 1; j <= V; j++) {
                arr[j] = new ArrayList<>();
            }

            for (int j = 0; j < E; j++) {
                int u = sc.nextInt();
                int v = sc.nextInt();

                if (u <= v) {
                    arr[u].add(v);
                } else {
                    arr[v].add(u);
                }
            }

            boolean isBreak = false;

            for (int j = 1; j <= V; j++) {
                if (grp[j] == 0) {
                    isBreak = DFS(E + j, j, arr, grp);
                    if (isBreak) {
                        break;
                    }
                }
            }

            System.out.println(isBreak ? "YES" : "NO");
        }

    }

    private static boolean DFS(int start, int v, ArrayList<Integer>[] arr, int[] grp) {

        grp[v] = start;

        for (int a : arr[v]) {
            if (grp[a] == 0) {
                DFS(start, a, arr, grp);
            } else if (grp[a] != start) {
                return true;
            }
        }

        return false;
    }
}


```
