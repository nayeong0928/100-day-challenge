# 회의실 배정

## 문제 분석

- N개의 회의실
- 회의 개수의 범위 : 1 ≤ N ≤ 100,000
- 사용할 수 있는 회의의 최대 개수 출력

## 풀이 

- 종료 시간이 빠른 회의부터 정렬
- 종료 시간이 같은 경우에는 시작 시간이 빠른 순서로 정렬
- end: 이전 회의가 종료되고, 새로운 회의가 시작 가능한 시간

### 구현

```java

import java.util.Arrays;
import java.util.Comparator;
import java.util.Scanner;

public class sol1931 {

    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);
        int N = sc.nextInt();
        int[][] arr = new int[N][2];

        for (int i = 0; i < N; i++) {
            for (int j = 0; j < 2; j++) {
                arr[i][j] = sc.nextInt();
            }
        }

        System.out.println(solution(arr));

    }

    private static int solution(int[][] arr) {

        Arrays.sort(arr, new Comparator<int[]>() {
            @Override
            public int compare(int[] o1, int[] o2) {
                if (o1[1] == o2[1]) {
                    return o1[0] - o2[0];
                }

                return o1[1] - o2[1];
            }
        });

        int end = -1;
        int cnt = 0;

        for (int i = 0; i < arr.length; i++) {
            if (arr[i][0] >= end) {
                end = arr[i][1];
                cnt++;
            }
        }

        return cnt;
    }

}

```
