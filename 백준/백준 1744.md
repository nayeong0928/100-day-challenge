# 수 묶기 

## 분석

- 수열의 두 수를 묶음 -> 묶은 수는 곱한다.
- 두 수는 위치 상관없이 묶을 수 있음.
- 수를 묶지 않아도 됨 -> 묶지 않은 수는 더한다.
- 수열의 수는 -1000 ~ 1000 범위.
- 수열의 개수는 50보다 작은 자연수.
- 묶은 결과가 최대인 수를 구해야 한다.


## 풀이

### 수열의 수가 가질 수 있는 경우

- 양수인 경우: 큰 수부터 시작해서 두 수를 곱함.
- 음수인 경우: 작은 수부터 시작해서 두 수를 곱함. (음수 둘을 곱하면 양수가 되기 때문)
- 0인 경우: 음수 한 개와 0만 남은 경우, 곱하면 결과값에서 빼는 수가 0이 되므로 곱함. 나머지 경우는 더함.
- 1인 경우: 곱하는 것보다 더하는 것이 결과값에 더해지는 수가 크기 때문에 무조건 더함. 

### 구현

```java

import java.util.Comparator;
import java.util.PriorityQueue;
import java.util.Scanner;

public class sol1744 {

    public static void main(String[] args) {
        PriorityQueue<Integer> q1 = new PriorityQueue<>(Comparator.reverseOrder());
        PriorityQueue<Integer> q2 = new PriorityQueue<>();
        Scanner sc = new Scanner(System.in);
        int N = sc.nextInt();

        for (int i = 0; i < N; i++) {
            int n = sc.nextInt();
            if (n > 0) {
                q1.add(n);
            } else {
                q2.add(n);
            }
        }

        int ans = 0;

        // 양수인 경우
        while (!q1.isEmpty()) {
            int num = q1.poll();
            if (!q1.isEmpty() && q1.peek() != 1) {
                num *= q1.poll();
            }
            ans += num;
        }

        // 음수인 경우
        while (!q2.isEmpty()) {
            int num = q2.poll();
            if (!q2.isEmpty()) {
                num *= q2.poll();
            }
            ans += num;
        }

        System.out.println(ans);

    }
}

```
