# 여행 가자

## 문제 분석

- 같은 도시를 두번 이상 방문 가능
- 도시 연결은 양방향 연결
- 여행 계획이 모두 연결된 도시인지 여부 리턴하기

## 풀이

- 도시 연결 정보가 입력될 때마다 union()으로 연결해줌
- 값이 더 작은 도시를 대표 도시로 삼음
- 도시를 연결할 때에는 두 도시의 대표 도시(parent 배열)끼리 연결해줌
- 대표 도시가 바뀌었을 때 기존의 대표 도시 정보는 find()에서 갱신

```java

import java.util.Scanner;

public class P1976 {

    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);

        int N = sc.nextInt();
        int M = sc.nextInt();

        int[] parent = new int[N + 1];
        for (int i = 1; i <= N; i++) {
            parent[i] = i;
        }

        for (int i = 1; i <= N; i++) {
            for (int j = 1; j <= N; j++) {
                int n = sc.nextInt();
                if (n == 1) {
                    union(i, j, parent);
                }
            }
        }

        int[] plan = new int[M];

        for (int i = 0; i < M; i++) {
            plan[i] = sc.nextInt();
        }

        int grp = parent[plan[0]];
        for (int i = 0; i < M; i++) {
            if (grp != parent[plan[i]]) {
                System.out.println("NO");
                return;
            }
        }

        System.out.println("YES");

    }

    private static void union(int a, int b, int[] parent) {
        int A = find(a, parent);
        int B = find(b, parent);

        if (A < B) {
            parent[B] = A;
        } else {
            parent[A] = B;
        }
    }

    private static int find(int a, int[] parent) {

        if (parent[a] == a)
            return a;

        parent[a] = find(parent[a], parent);
        return parent[a];
    }

}


```
