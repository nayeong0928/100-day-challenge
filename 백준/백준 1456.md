# 거의 소수

## 문제 분석

- 거의소수: 소수의 N제곱인 수
- A보다 크거나 같고, B보다 작거나 같은 거의소수의 개수 구하기
- 1 ≤ A ≤ B ≤ 10^14

## 풀이 (진행중)

- 거의소수가 소수의 2제곱으로 오독
- 소수를 구한 다음, 범위 안에서 2제곱 이상의 수도 체크해야 한다

```java

import java.util.Scanner;

public class sol1456 {

    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);
        long A = sc.nextLong();
        long B = sc.nextLong();

        int start = (int) Math.floor(Math.sqrt(A));
        int end = (int) Math.ceil(Math.sqrt(B));

        while (Math.pow(start, 2) < A) {
            start++;
        }

        while (Math.pow(end, 2) > B) {
            end--;
        }

        System.out.println(start + ", " + end);

        boolean[] arr = new boolean[end + 1];
        arr[1] = true;

        for (int i = 2; i < arr.length; i++) {
            for (int j = 2; j < arr.length; j++) {
                int n = i * j;
                if (n >= arr.length)
                    break;
                arr[n] = true;
            }
        }

        int cnt = 0;
        for (int i = start; i <= end; i++) {
            if (!arr[i]) {
                cnt++;
            }
        }

        System.out.println(cnt);

    }
}

```
