# 최단경로

## 문제 분석

- 정점의 개수 V와 간선의 개수 E
- 1 ≤ V ≤ 20,000, 1 ≤ E ≤ 300,000
- (u, v, w) : u에서 v로 가는 가중치 w인 간선이 존재함 -> 방향 그래프
- 시작점 K에서부터 i번째 줄에 i번 정점으로 가는 경로의 최소값 출력
- 경로가 존재하지 않는 경우에는 INF를 출력

## 풀이

- dist: 경로 최소값, INF로 초기화
- queue: 해당 정점을 거쳐서 다른 정점으로 가는 경우 고려하는 용도
- arr: 그래프, (u, v, w)를 arr[u] -> Vertex(v,w)로 나타냄

```java

import java.util.ArrayList;
import java.util.Arrays;
import java.util.LinkedList;
import java.util.Queue;
import java.util.Scanner;

public class P1753 {

    public static void main(String[] args) {

        final int INF = 400000;
        Scanner sc = new Scanner(System.in);
        int V = sc.nextInt();
        int E = sc.nextInt();
        int K = sc.nextInt();

        ArrayList<Vertex>[] arr = new ArrayList[V + 1];
        int[] dist = new int[V + 1];
        boolean[] visited = new boolean[V + 1];

        Arrays.fill(dist, INF);
        dist[K] = 0;

        for (int i = 1; i <= V; i++) {
            arr[i] = new ArrayList<>();
        }

        for (int i = 0; i < E; i++) {
            int u = sc.nextInt();
            int v = sc.nextInt();
            int w = sc.nextInt();

            arr[u].add(new Vertex(v, w));
        }

        Queue<Vertex> queue = new LinkedList<>();
        queue.add(new Vertex(K, 0));

        while (!queue.isEmpty()) {
            Vertex q = queue.poll();
            if (visited[q.v])
                continue;

            for (Vertex a : arr[q.v]) {
                if (dist[a.v] > (dist[q.v] + a.e)) {
                    dist[a.v] = dist[q.v] + a.e;
                    queue.add(a);
                }
            }

            visited[q.v] = true;
        }

        for (int i = 1; i <= V; i++) {
            System.out.println(dist[i] == INF ? "INF" : dist[i]);
        }
    }

    static class Vertex {
        int v;
        int e;

        public Vertex(int v, int e) {
            this.v = v;
            this.e = e;
        }
    }

}


```
