# 거짓말

## 문제 분석

- 지민이가 거짓말쟁이가 되지 않는 파티 개수의 최대값
- N, M은 50 이하의 자연수이고, 진실을 아는 사람의 수는 0 이상 50 이하의 정수, 각 파티마다 오는 사람의 수는 1 이상 50 이하의 정수

## 풀이

- union-find 로 과장된 이야기를 할 수 있는 그룹, 진실을 말해야 하는 그룹으로 분리
- hash: 진실을 아는 사람 그룹 대표 저장, party[i]를 순회하면서 이 값을 parent로 가진 그룹 사람이 있는지 체크

```java

import java.util.HashSet;
import java.util.Scanner;

public class P1043 {

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int N = sc.nextInt();
        int M = sc.nextInt();

        int[] parent = new int[N + 1];
        int[][] party = new int[M][];

        for (int i = 1; i < parent.length; i++) {
            parent[i] = i;
        }

        int num = sc.nextInt();
        int[] know = new int[num];

        for (int i = 0; i < num; i++) {
            know[i] = sc.nextInt();
        }

        for (int i = 0; i < M; i++) {
            int n = sc.nextInt();
            party[i] = new int[n];

            for (int j = 0; j < n; j++) {
                party[i][j] = sc.nextInt();
                union(party[i][0], party[i][j], parent);
            }
        }

        HashSet<Integer> hash = new HashSet<>();

        for (int i = 0; i < know.length; i++) {
            int p = find(know[i], parent);
            hash.add(p);
        }

        int cnt = 0;

        for (int i = 0; i < party.length; i++) {
            if (!same(party[i], hash, parent)) {
                cnt++;
            }
        }
        System.out.println(cnt);
    }

    private static void union(int a, int b, int[] parent) {
        int A = find(a, parent);
        int B = find(b, parent);

        if (A < B) {
            parent[B] = A;
        } else {
            parent[A] = B;
        }
    }

    private static int find(int a, int[] parent) {
        if (parent[a] == a)
            return a;

        parent[a] = find(parent[a], parent);
        return parent[a];
    }

    private static boolean same(int[] party, HashSet<Integer> hash, int[] parent) {
        for (int i = 0; i < party.length; i++) {
            int p = find(party[i], parent);
            if (hash.contains(p)) {
                return true;
            }
        }

        return false;
    }
}

```
